ADC
===

ADCREF
------

:type: enum
:size: 1 byte

ADC reference voltage source. Values:

* 0: AREF
* 1: AVCC
* 3: Internal 1.1V

ADCUREF
-------

:type: float
:size: 4 bytes

Reference voltage, volts.

MAX7456
=======

VMODE
-----

:type: enum
:size: 1 byte

OSD composite video mode. Values:

* 0: PAL
* 1: NTSC
